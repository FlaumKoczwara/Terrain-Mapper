uint8_t RFID_add_card_to_database ( void  )

Dodaje kod zapisany w karcie do bazy danych.

Zwraca
Zwraca status powodzenia operacji 0 => karty nie znaleziono 1 => karta jest już w bazie 2 => zapisano karte Jeśli karty nie ma w bazie danych jest ona do niej dodawana na wolnej pozycji Jeśli karta przyłożona do czujnika jest już zapisana w bazie danych.
uint8_t RFID_delete_card_from_database ( void  )

Usuwa z bazy danych kartę przyłożoną do czytnika.

Zwraca
0 => brak karty w bazie bądź karta nie została odczytana 1 => przyłożona karta zostala z bazy usunieta
uint8_t RFID_get_serial_number ( uint8_t *  serial_number)

Funkcja odczytująca unikalny numer seryjny karty.

Parametry
uint8_t*serial_number - wskaźnik do tablicy o wymiarze minimum 5, w której zapisujemy odczytany numer seryjny
Zwraca
Zwraca status powodzenia wykonanej operacji: 1 - znaleziono kartę i odczytano z niej numer seryjny 0 - karty nie znaleziono bądź wystąpił błąd transmisji
void RFID_init ( void  )

Funkcja inicjalizująca piny GPIO, interfejs SPI oraz moduł RC522.

Ostrzeżenie
Musi zostać wywołana przed pętlą nieskończoną w funkcji main()
uint8_t RFID_search_database ( uint8_t *  serial_number)

Funkcja szuka podanego numeru seryjnego w bazie danych.

Parametry
uint8_t*serial_number - wskaźnik do tablicy której numer seryjny chcemy sprawdzić
Zwraca
Zwraca status powodzenia wykonanej operacji: 0 => w bazie nie ma takiego numeru seryjnego 0< => numer seryjny jest zapisany w danej komórce bazy danych np.(zwrócenie 5 oznacza zapisanie karty na 5 komórce w bazie danych)
uint8_t RFID_show_database ( uint8_t  tab[CARD_ARRAY_SIZE][5])

Pokazuje zawartość bazy danych. Kopiuje zapisane numery seryjne z bazy danych do tablicy użytkownika.

Zwraca
Zwraca ile slotów jest zajętych
