uint8_t RC522_to_card ( uint8_t  cmd,
uint8_t *  send_data,
uint8_t  send_data_len,
uint8_t *  back_data,
uint32_t *  back_data_len 
)

Funkcja realizująca komunikację kartą MIFARE poprzez RC522.

Parametry
uint8_tcmd - komenda jaką ma zrealizować RC522
uint8_t*send_data - wskaźnik do danych które chcemy wysłać do karty (mogą to być komendy obsługiwane przez kartę np PIC_REQALL
uint8_tsend_data_lenght - ile danych wysyłamy do karty
uint8_t*back_data - wskaźnik do tablicy/zmiennej w której zapisujemy dane odebrane z karty (UID, zapisane wcześniej dane)
uint32_t*back_data_lenght - ilość danych jaka znajduje się w buforze FIFO, i którą musimy odebrać
Zwraca
Zwraca status powodzenia wykonanej operacji

Realizuje podstawową komunikację z kartą typu MIFARE. Możliwe jest wysyłanie komend i danych do karty RFID. Funkcja umożliwia również odbiór danych z karty.

uint8_t RFID_add_card_to_database ( void  )

Dodaje kod zapisany w karcie do bazy danych.

Zwraca
Zwraca status powodzenia operacji 0 => karty nie znaleziono 1 => karta jest już w bazie 2 => zapisano karte Jeśli karty nie ma w bazie danych jest ona do niej dodawana na wolnej pozycji Jeśli karta przyłożona do czujnika jest już zapisana w bazie danych.
uint8_t RFID_delete_card_from_database ( void  )

Usuwa z bazy danych kartę przyłożoną do czytnika.

Zwraca
0 => brak karty w bazie bądź karta nie została odczytana 1 => przyłożona karta zostala z bazy usunieta
uint8_t RFID_get_serial_number ( uint8_t *  serial_number)

Funkcja odczytująca unikalny numer seryjny karty.

Parametry
uint8_t*serial_number - wskaźnik do tablicy o wymiarze minimum 5, w której zapisujemy odczytany numer seryjny
Zwraca
Zwraca status powodzenia wykonanej operacji: 1 - znaleziono kartę i odczytano z niej numer seryjny 0 - karty nie znaleziono bądź wystąpił błąd transmisji

uint8_t RFID_read_card ( uint8_t *  serial_out)

Funkcja odczytująca unikalny numer seryjny karty.

Parametry
uint8_t*serial_out - wskaźnik do tablicy o wymiarze minimum 5, w której zapisujemy wynik pomiarów
Zwraca
Zwraca status powodzenia wykonanej operacji UID - Unique ID ( unikalny numer seryjny )
Ostrzeżenie
Przed wywołaniem funkcji karta musi zostać aktywowana wywołaniem funkcji RFID_request(...)
uint8_t RFID_request ( uint8_t *  tag_type)

Funkcja inicjuje komunikacje z kartą znajdującą się w zasięgu anteny czytnika.

Parametry
uint8_t*tag_type - wskaźnik do tablicy do której ma zostać napisany typ karty którą odczytujemy
Zwraca
Zwraca status powodzenia komunikacji

Funkcja wysyła rządanie nawiązania kounikacji do kart które znajdują się w polu magnetycznym czytnika. Karta odsyła nam jako informację zwrotną jakim jest typem

uint8_t RFID_search_database ( uint8_t *  serial_number)

Funkcja szuka podanego numeru seryjnego w bazie danych.

Parametry
uint8_t*serial_number - wskaźnik do tablicy której numer seryjny chcemy sprawdzić
Zwraca
Zwraca status powodzenia wykonanej operacji: 0 => w bazie nie ma takiego numeru seryjnego 0< => numer seryjny jest zapisany w danej komórce bazy danych np.(zwrócenie 5 oznacza zapisanie karty na 5 komórce w bazie danych)

Zwraca
